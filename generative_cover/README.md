## Windows

`cairo` ist wahrscheinlich nicht auf Windows Rechnern installiert. Deshalb klappt die Umwandlung von svg -> png nicht. In der `config.toml` deshalb auf *no_png* umstellen. 

Mit dem Skript `svg_to_png.ps1` können die Dateien dann umgewandelt werden.
